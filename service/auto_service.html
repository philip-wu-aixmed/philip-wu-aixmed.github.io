Operation Steps for Model Inference Services

  • Cytotechnologists copy WSI files into dedicated folders
    • - dedicated folders are located in \\SS7922\autoinference
    - cytotechnologists copy WSI filese into specific folder for specific purpose
    • folder {model version}_{decart versoin} is for model inference using dedicated version of AI model
    • folder tungs is dedicated for urine slides from TUNGS
    • folder ShihWen is for specified purpose defined by cytotechnologist
  • Execute model inference process
    • - move WSI files from dedicated folder to the working folder in inference service PC \\SATURN-WIN
    - execute python programs to model inference on working folder
    .\done\*.med: image files for viewing by CytoInsights
    .\done\*.aix: analyzed metadata by model
    dc-results-{timestamp}.csv: summary of analyzed metadata for cytotechnologists review
  • Notify cytotechnologists the analyzed images and metadata files
  • upload analyzed images to dedicated cloud bucket if customer service
    • - for TUNGS
    set cloud bucket to TOKYO\tungsiac by executing decart -s in CMD window
    move *.med and *.aix files into DeCart watch folder
    - for SCMH
    login Google Drive using SCMH account scmh01.c@gmail.com
    copy *med and *.aix files into Google Drive
  • backup WSI files and analyzed images into NAS
    • - backup WSI files and *.med/*.aix files into NAS \\aixmedstorage\\Auto Inference Backup
  • maintain the inference results
    • - append dc-results-{timestamp}.csv into summary file for cytotechnologist or customers
    • summary for TUNGS
    • model inference summary
